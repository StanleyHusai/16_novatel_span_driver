# 16_novatel_span_driver
Customized Deriver for SPAN-CPT

## Package summary

*16_novatel_span_driver* is a customized ROS node used to collect data from SPAN-CPT.

## Running the demo

`$ sudo chmod -R 777 /dev/ttyUSB0`
`$ ./runspan_cpt.sh`
